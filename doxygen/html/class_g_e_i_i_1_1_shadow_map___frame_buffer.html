Detailed Description

Spécialisation de OGL_FrameBuffer pour l'utilisation des lampes.

La shadow map est stocquée dans le texture de profondeur (GL_DEPTH_ATTACHMENT).
Aucune texture n'est attachée à la couleur.
La lecture et l'écriture vers le FBO sont désactivées.
