Permet de construire un modèle 3D, que l'on pourra dessiner avec OpenGL.

Un contexte OpenGL valide doit être actif lors de toute la durée de vie de cet objet (destruction incluse)

Parameters
vertice: Tableau contenant toutes les position des points (1 point = 3 cases (x,y,z))
textUV: Tableau contenant toutes les coordonnées UV des points (1 point = 2 cases(U,V))
normals: Tableau contenant toutes les normales des points (1 point = 3 cases (x,y,z))
indices: Tableau contenant la liste des indices permettant de définir l'ordre des points lors du dessin.
drawMode: Mode de dessin OpenGL. Doit être une des constantes suivantes :
  • GL_POINTS
  • GL_LINE_STRIP
  • GL_LINE_LOOP
  • GL_LINES
  • GL_LINE_STRIP_ADJACENCY
  • GL_LINES_ADJACENCY
  • GL_TRIANGLE_STRIP
  • GL_TRIANGLE_FAN
  • GL_TRIANGLES
  • GL_TRIANGLE_STRIP_ADJACENCY
  • GL_TRIANGLES_ADJACENCY
