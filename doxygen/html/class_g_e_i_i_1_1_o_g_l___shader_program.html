Detailed Description

Gère les objets OpengGL de type "ShaderProgram".

Les attributs de vertices utilisés dans le code GLSL sont les suivants :
"in_Vertex" (location 0) : vec3 comportant les positions des points du modèle 3D
"in_Normal" (location 1) : vec3 contenant la normale de chaque point.
"in_UV" (location 2) : vec2 contenant les coordonnées de textures de chaque point.
"in_Tangent" (location 3) : vec3 contenant les vecteurs tangents de chaque point. (utilisé pour le normal mapping)
"in_BiTangent" (location 4) : vec3 contenant les vecteurs bitangents de chaque point. (utilisé pour le normal mapping)

Crée un Shader à partir de code GLSL.

Lance une exception si il y a une erreur lors de la création du shader.

Parameters
vShader:
  • isFile = true -> Nom du fichier contenant le code GLSL du vertex shader.
  • isFile = false -> Code GLSL du vertex shader.
fShader:
  • isFile = true -> Nom du fichier contenant le code GLSL du fragment shader.
  • isFile = false -> Code GLSL du fragment shader.
isFile: indique si les chaines de caracères sont des noms de fichiers (true) ou du code GLSL (false).
